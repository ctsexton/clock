# Dashboard

> A simple dashboard app with responsive UI. Vue.js, Vuex & Nuxt.

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for GitHub pages
$ npm run build:gh-pages
$ npm run deploy
```
